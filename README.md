## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Utility-first styling
- **shadcn/ui** - High-quality UI components
- **Geist Font** - Beautiful typography from Vercel
- **Lucide React** - Icon library

## Sections

1. **Intro** - Name, availability status, location, and focus areas
2. **Selected Work** - Timeline of projects with tags and descriptions
3. **Recent Thoughts** - Blog posts and articles
4. **Connect** - Email and social media links
5. **Footer** - Copyright and additional links

## Development

```bash
npm install
npm run dev
```

## Customization

Edit the content in each section component:
- `src/components/IntroSection.jsx` - Personal info and focus areas
- `src/components/WorkSection.jsx` - Projects and work history
- `src/components/ThoughtsSection.jsx` - Blog posts and articles
- `src/components/ConnectSection.jsx` - Contact information
